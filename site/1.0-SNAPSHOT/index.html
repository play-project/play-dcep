About play-dcep

The PLAY Distributed Complex Event Processing (DCEP) component.

Project Modules

This project has declared the following modules:

Name Description
play-platformservices-core The core of the PLAY Patformservices component.
play-platformservices-api The API artifacts for PLAY Platformservices.
play-platformservices-bdpl The grammar for Big Data Processing Language (BDPL).
play-platformservices-querydispatcher The PLAY Query Dispatch service.
play-platformservices-eventvalidation Code to check RDF events for validity in terms of minimum number of properties, etc.
play-dcep-api DCEP API artifacts.
play-dcep-core The DCEP core component.
play-dcep-distributedetalis The Distributed Etalis component which represents the event processing agents inside DCEP.
play-dcep-dcepmanager The DCEP Manager component responsible for nonfunctional aspects of controlling DCEP.
Distributed Complex Event Processing (DCEP) The DCEP distribution: a complete executable package of DCEP and Platformservices with all dependencies.
